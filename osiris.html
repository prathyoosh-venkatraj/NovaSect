    WHY OUTPUTS VARY BETWEEN RUNS
    Osiris draws fresh random numbers on every simulation. At the default path count, P50/P95/P05 lines can shift ±1–2% run-to-run — this is normal sampling variance, not a model error.

    HI-FI mode raises the path count to tighten that variance:
    50K paths — ±0.5% drift 100K paths — ±0.1% drift 250K paths — near-deterministic
